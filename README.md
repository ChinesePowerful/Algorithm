###简介
- **数据结构** 是由「相互之间存在着一种或多种关系的数据元素的集合」和「该集合中数据元素之间的关系」组成的
- **算法** 是指解题方案的准确而完整的描述，是一系列解决问题的清晰指令，算法代表着用系统的方法描述解决问题的策略机制
